# MorphoCell

MorphoCell is a Python library for morphomeric analysis of 3D cell images.

## Installation

Tested with:
* Python 3.9-11
* Ubuntu 18.04
* CUDA 11.2-12.1

The project can be installed from source as following:

```bash
git clone https://github.com/alxndrkalinin/morphocell.git
cd morphocell
pip install -e .[all]
```
